# CoffeeOrderingSystem
The Coffee Ordering System allows users to simulate basic coffee shop operations using python. Customers can view the menu, place coffee orders by selecting the drink type, size and toppings and then receive an invoice with a cost breakdown.
### Features
- Menu-Driven Interface
- Coffee selection with multiple sizes
- Up to 3 free toppings per drink, with additional toppings costing $1
- Provides detailed summary of the order and cost
